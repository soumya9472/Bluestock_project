# Mutual Fund Analytics Project

## Overview
This project analyzes mutual fund investment data to identify investment patterns, top-performing states, transaction trends, and investor behavior.

## Tech Stack
- Python
- Pandas
- SQLite
- Power BI

## Project Workflow
1. Data Ingestion
2. Data Cleaning
3. SQL Database Creation
4. Exploratory Data Analysis (EDA)
5. Power BI Dashboard

## Dashboard Highlights
- Total Investment Amount
- Total Transactions
- Total Investors
- Top 10 States by Investment Amount
- Transaction Type Distribution
- Gender-wise Investment Distribution
